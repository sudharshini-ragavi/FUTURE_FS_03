# 🍽️ Spice & Bliss – Restaurant Website

## 📌 Project Overview
This project is developed as part of Future Interns Task 3 (FS Track).

Spice & Bliss is a modern restaurant website designed to showcase:
- South Indian, North Indian & Continental dishes
- Attractive ambience
- Online table booking system
- Customer reviews

## 🚀 Features
- Animated modern UI
- Interactive navigation
- Image gallery with full-screen preview
- Booking form (Name, Date, Time)
- Local storage reviews
- Responsive design

## 🎯 Objective
To build a real-world business website that:
- Improves online presence
- Attracts customers
- Builds trust
